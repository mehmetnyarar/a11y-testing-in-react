# A11y Testing with Storybook

```sh
npx sb init --builder webpack5
yarn add webpack @storybook/jest -D
yarn add @storybook/preset-scss css-loader sass sass-loader style-loader -D
yarn add  @storybook/addon-a11y -D
```
